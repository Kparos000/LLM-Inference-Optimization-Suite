# Finance 10,000-Record Dataset EDA

This folder is the public Finance entry point for the 10,000-record dataset EDA.
Open `finance_eda.html` to review Finance-specific prompt, gold/eval, KB,
evidence-reuse, workload, and term-analysis views.

The global cross-vertical EDA lives under:

```text
data/generated/dataset_10000/
```

This folder is generated by:

```powershell
python scripts/phase2/explore_phase2a_promoted_dataset.py --dataset-root data/scaleup_2000_full --write-report
```

This is a convenience mirror of Finance artifacts from the global dataset EDA;
it is not a separate benchmark dataset.
